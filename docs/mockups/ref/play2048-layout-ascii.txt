STRUCTURE SUMMARY
-----------------
1. Header: game title (2048) + score + best score + New game button.
2. Tagline: one line of instruction (optional).
3. Main: 4x4 grid only; no sidebar, no how-to modal by default.
4. Input: keyboard (arrows / WASD) or touch swipe; no visible control UI.
5. Footer: Privacy Policy link only.

TAKEAWAYS FOR URSA MINOR
------------------------
- Ultra-minimal game page: title, score, one CTA, grid. No nav in header if single-game site.
- Score + Best score in header is standard for puzzle/arcade games.
- 4x4 grid is the only content; no rules block unless expandable.
- Footer can be a single link (e.g. Privacy) for third-party reference; Ursa Minor would use horizon footer.

STRUCTURE SUMMARY
-----------------
1. Header: logo + nav (Home, Games, About) - sticky, glass
2. Starfield: full bleed background (night sky motif)
3. Hero: logo lockup + tagline (max 6 words) + two CTAs (Play, Browse)
4. Games: optional heading + visual-first card grid (motif + title)
5. Footer: horizon line, back-to-top, copyright

TAKEAWAYS APPLIED
-----------------
- Neal.fun: Minimal chrome, grid-first, no section bloat
- NYT: Clear hero + game grid structure, but stripped of editorial/learn-more
- Papergames: Rejected value-props and feature blocks (marketing tone)
- DESIGN_BIBLE: Logo + tagline + CTAs; visual-first cards; calm, no noise
- Peaceful night sky: Starfield, horizon footer, constellation motif
- 960px max content width (section class)
- Single-word nav labels
